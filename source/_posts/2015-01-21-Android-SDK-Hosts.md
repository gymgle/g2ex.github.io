```ini
# Android SDK
74.125.113.121 developer.android.com
203.208.46.146 dl.google.com
203.208.46.146 dl-ssl.google.com
# End Android SDK
```

Android SDK Manager `Tools` -> `Options`，勾选 `Force https://... sources to be fetched using http://...`
